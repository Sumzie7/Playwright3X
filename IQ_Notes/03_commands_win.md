# VS Code Commands - Windows

## General Commands

| Command | Shortcut | Description |
|---------|----------|-------------|
| Command Palette | `Ctrl + Shift + P` | Open command palette to run any command |
| Quick Open | `Ctrl + P` | Quickly open a file |
| New Window | `Ctrl + N` | Open a new VS Code window |
| Close Window | `Ctrl + W` | Close the current window |
| Settings | `Ctrl + ,` | Open user settings |
| Keyboard Shortcuts | `Ctrl + K Ctrl + S` | Open keyboard shortcuts reference |

## File Operations

| Command | Shortcut | Description |
|---------|----------|-------------|
| New File | `Ctrl + N` | Create a new file |
| Open File | `Ctrl + O` | Open a file dialog |
| Open Folder | `Ctrl + K Ctrl + O` | Open a folder |
| Save | `Ctrl + S` | Save the current file |
| Save All | `Ctrl + Alt + S` | Save all open files |
| Close Tab | `Ctrl + W` | Close the current tab |
| Close All Tabs | `Ctrl + K Ctrl + W` | Close all tabs |
| Reopen Closed Tab | `Ctrl + Shift + T` | Reopen the last closed tab |

## Editing

| Command | Shortcut | Description |
|---------|----------|-------------|
| Undo | `Ctrl + Z` | Undo last action |
| Redo | `Ctrl + Y` | Redo last undone action |
| Cut | `Ctrl + X` | Cut selected text |
| Copy | `Ctrl + C` | Copy selected text |
| Paste | `Ctrl + V` | Paste text from clipboard |
| Select All | `Ctrl + A` | Select all content |
| Find | `Ctrl + F` | Open find dialog |
| Replace | `Ctrl + H` | Open find and replace dialog |
| Find Next | `F3` or `Ctrl + G` | Find next match |
| Find Previous | `Shift + F3` or `Ctrl + Shift + G` | Find previous match |
| Add Line Comment | `Ctrl + /` | Comment/uncomment line |
| Add Block Comment | `Shift + Alt + A` | Comment/uncomment block |
| Delete Line | `Ctrl + Shift + K` | Delete entire line |
| Insert Line Below | `Ctrl + Enter` | Insert new line below |
| Insert Line Above | `Ctrl + Shift + Enter` | Insert new line above |
| Move Line Down | `Alt + Down` | Move line down |
| Move Line Up | `Alt + Up` | Move line up |
| Duplicate Line | `Shift + Alt + Down` | Duplicate line |
| Go to Line | `Ctrl + G` | Go to specific line number |

| Command | Shortcut | Description |
|---------|----------|-------------|

| Shrink Selection | `Shift + Alt + Left` | Shrink selection |
| Go to Definition | `F12` | Jump to definition |
| Peek Definition | `Ctrl + Shift + F10` | Preview definition without jumping |
| Go to Symbol | `Ctrl + Shift + O` | Go to symbol in file |
| Go to Symbol Global | `Ctrl + T` | Search for symbol across workspace |
| Next Error/Warning | `F8` | Jump to next error |
| Previous Error/Warning | `Shift + F8` | Jump to previous error |
