# VS Code Commands - Mac

## General Commands

| Command | Shortcut | Description |
|---------|----------|-------------|
| Command Palette | `Cmd + Shift + P` | Open command palette to run any command |
| Quick Open | `Cmd + P` | Quickly open a file |
| New Window | `Cmd + N` | Open a new VS Code window |
| Close Window | `Cmd + W` | Close the current window |
| Settings | `Cmd + ,` | Open user settings |
| Keyboard Shortcuts | `Cmd + K Cmd + S` | Open keyboard shortcuts reference |

## File Operations

| Command | Shortcut | Description |
|---------|----------|-------------|
| New File | `Cmd + N` | Create a new file |
| Open File | `Cmd + O` | Open a file dialog |
| Open Folder | `Cmd + K Cmd + O` | Open a folder |
| Save | `Cmd + S` | Save the current file |
| Save All | `Cmd + Alt + S` | Save all open files |
| Close Tab | `Cmd + W` | Close the current tab |
| Close All Tabs | `Cmd + K Cmd + W` | Close all tabs |
| Reopen Closed Tab | `Cmd + Shift + T` | Reopen the last closed tab |

|---------|----------|-------------|

| Redo | `Cmd + Shift + Z` | Redo last undone action |
| Cut | `Cmd + X` | Cut selected text |
| Copy | `Cmd + C` | Copy selected text |
| Paste | `Cmd + V` | Paste text from clipboard |
| Select All | `Cmd + A` | Select all content |
| Find | `Cmd + F` | Open find dialog |
| Replace | `Cmd + H` | Open find and replace dialog |
| Find Next | `Cmd + G` | Find next match |
| Find Previous | `Cmd + Shift + G` | Find previous match |
| Add Line Comment | `Cmd + /` | Comment/uncomment line |
| Add Block Comment | `Cmd + Alt + /` | Comment/uncomment block |
| Delete Line | `Cmd + Shift + K` | Delete entire line |
| Insert Line Below | `Cmd + Enter` | Insert new line below |
| Insert Line Above | `Cmd + Shift + Enter` | Insert new line above |
| Move Line Down | `Alt + Down` | Move line down |
| Move Line Up | `Alt + Up` | Move line up |
| Duplicate Line | `Alt + Shift + Down` | Duplicate line |
| Go to Line | `Cmd + G` | Go to specific line number |

| Command | Shortcut | Description |
|---------|----------|-------------|

| Shrink Selection | `Shift + Alt + Left` | Shrink selection |
| Go to Definition | `F12` | Jump to definition |
| Peek Definition | `Cmd + Alt + F12` | Preview definition without jumping |
| Go to Symbol | `Cmd + Shift + O` | Go to symbol in file |
| Go to Symbol Global | `Cmd + T` | Search for symbol across workspace |
| Next Error/Warning | `F8` | Jump to next error |
| Previous Error/Warning | `Shift + F8` | Jump to previous error |
